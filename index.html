3. Voard (Video Board)

🡺
🡸

SpringBoot / JPA / QueryDsl 활용하여 만든 유튜브 링크 저장 게시판 입니다. (Video Board)

로그인과 회원가입은 SpringSecurity를 활용하여 구현하였으며 유저 테이블의 Column(Role)에 따라서 권한을 부여하였습니다.

링크를 input 창에 입력하고 추가 버튼을 누르게 되면 카테고리 / 완료 / 취소와 같은 3개의 버튼과 입력한 Youtube 영상이 생성됩니다.

그 이후 카테고리를 설정 후 완료창을 누르게 되면 DB에 저장되게 됩니다.

페이징의 경우 Intersection Observer를 사용하여 무한스크롤 형식으로 구현하였으며 자바스크립트의 page 변수에 따라서 ajax로 데이터를 요청하며, 데이터가 없는 경우 Observer target이 노출되더라도 데이터를 요청하지 않습니다.

페이징 쿼리는 offset과 skip을 사용하여 데이터를 불러오고 있습니다.

깃허브 - https://github.com/gsbyo/Voard
